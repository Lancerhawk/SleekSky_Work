Deploy Content

Automate workflows when new content versions are created (e.g., rebuild a site or notify a team).

Add Action Choose a pre-configured integration:

  1. GitHub Action: Trigger a GitHub Actions workflow.
  2. GitHub Trigger: Send a repository dispatch event to any GitHub repo.
  3. Netlify Build: Rebuild your Netlify site with the latest content version.
  4. Custom Webhook: Send a POST request to your endpoint with payload details like version_tag and content_url.

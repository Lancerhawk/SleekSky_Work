Features

Flexible Content Modeling

Define pages, entries, blocks, and lists with custom fields. Build reusable content structures that adapt to your project needs with hierarchical organization and dynamic key-value pairs.

Static Site Generation

ld lightning-fast static sites using EJS templates and Tailwind CSS. Preview in real-time, download builds, or deploy directly to GitHub Pages, Netlify, and other platforms.

Access your content programmatically with development and production APIs. Manage environment tags, content versions, and deploy with confidence using our comprehensive API endpoints.

Multilingual & Multi-Tenant

Localize content for global audiences and customize experiences for different clients or organizational units. Manage translations and tenant-specific content versions with ease.

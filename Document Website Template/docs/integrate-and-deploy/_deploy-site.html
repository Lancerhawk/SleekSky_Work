Deploy Site

Deploy static sites built with SleekCMS to hosting platforms like GitHub Pages or Netlify.

Download Site Download a ZIP of your site’s code for a specific content version. Click **Download ** to get the build files (HTML, CSS, JS, and content JSON).

  • Use Case: Manual deployments or debugging build issues.

Add Action Automate deployments using integrations.

GitHub Push

  1. Push site builds directly to a GitHub repository.
  2. Action Name: A unique label (e.g., “Prod Deployment”).
  3. Personal Access Token: Generate a token with repo scope in GitHub Settings.
  4. Repository: Format org/repo (e.g., sleekcms/docs).
  5. Ref: Branch name (e.g., main). The action will commit the build to this branch.

Troubleshooting:

  1. Ensure the token has write access to the repository.
  2. Verify the branch exists before triggering the action.

Netlify Deploy Trigger Netlify deployments automatically.

  1. Netlify Token: Find this in Netlify Account Settings > Applications.
  2. Connect: Click to authenticate Netlify.
  3. Deploy: New site builds trigger automatically when content versions are published.
